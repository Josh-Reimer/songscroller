## Song Scroller
now you can doom scroll the christian hymnal song book.
congratulations.

i have not provided the song jpgs but if you download the christianhymnal from https://archive.org/details/christianhymnalc00chur and convert to jpgs, then rename them all to have names like songscroller345.jpg (replace number with consecutive numbers), then move them all to the img folder in the root of this project, you should be good to go.
